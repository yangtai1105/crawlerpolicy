    Google's open-source Agent Development Kit (ADK) for Python received recent updates including Custom Service Registration, a Rewind feature for sessions, and a new `AgentEngineSandboxCodeExecutor` for executing agent-generated code using Vertex AI Code Execution Sandbox API. Furthermore, the A2A (Agent-to-Agent) Protocol, which ADK for Java natively supports, announced on April 9, 2026, that it has surpassed 150 supporting organizations and achieved deep integration across Google, Microsoft, and AWS platforms, becoming a production-ready open standard for seamless agent-to-agent communication.

    On April 17, 2026, Google released A2UI v0.9, a framework-agnostic standard for declaring UI intent. This update includes an Agent SDK, new language features, and simplified transports, making it easier to build agents and integrate them with existing frontends. It also highlights the official launch of the A2A 1.0 protocol.

8.  **Microsoft Agent Framework 1.0 Released, Incorporating AutoGen**
    Microsoft announced the release of Agent Framework 1.0 on April 3, 2026. This framework unifies the enterprise foundations of Semantic Kernel with the orchestration capabilities of AutoGen, providing stable support for multi-agent orchestration patterns and declarative agents and workflows. It also includes support for the Model Context Protocol (MCP) and upcoming support for the A2A protocol.
