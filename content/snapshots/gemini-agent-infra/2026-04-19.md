Here's a compact digest of the most important distinct items regarding new features, integrations, and announcements from AI agent infrastructure vendors, focusing on protocol and bot-identity features, from the last 14 days (April 5, 2026, to April 19, 2026):

1.  **Anthropic Launches Claude Managed Agents**
    Anthropic announced Claude Managed Agents, a managed infrastructure solution designed to simplify the creation and deployment of AI agents for businesses. This platform handles underlying complexities such as state management, memory control, task orchestration, credential management, and scoped permissions, allowing enterprises to deploy agents more rapidly.

2.  **Anthropic Integrates Biometric ID Verification for Claude**
    Anthropic has added limited biometric identity verification capabilities to Claude through a partnership with Persona. This integration aims to enhance the security and trustworthiness of interactions with Claude agents by providing a mechanism for verifying user identity.

3.  **Anthropic Restricts Mythos Model Release and Launches Project Glasswing**
    Anthropic is limiting access to its Mythos Preview model due to its advanced ability to identify and exploit software vulnerabilities, demonstrating "superhuman hacker capabilities." Concurrently, Anthropic launched "Project Glasswing," a collaboration with major tech and cybersecurity firms to test the Mythos model for defensive cybersecurity applications, emphasizing the development of safeguards before broader deployment.

4.  **OpenAI Joins FIDO Alliance to Advance AI Agent Authentication**
    OpenAI has become a member of the FIDO Alliance, a passwordless authentication industry association, and joined its Board of Directors. OpenAI plans to contribute to evolving authentication standards for agentic intelligence, aiming to ensure AI agents are trustworthy, verified, and governed by user intent.

5.  **OpenAI Announces Next Evolution of Agents SDK**
    OpenAI revealed "The next evolution of the Agents SDK," indicating ongoing advancements in its toolkit for developing AI agents. This suggests upcoming improvements in how developers can build and integrate agents with OpenAI's platforms.

6.  **OpenAI Expands Access to GPT-5.4-Cyber with Trust Signals**
    OpenAI announced the broader availability of its GPT-5.4-Cyber model, a chatbot fine-tuned for cybersecurity capabilities. The company emphasized that access will be granted to vetted security vendors, organizations, and researchers, with internal safeguards, know-your-customer verification, and "trust signals" in place to prevent misuse.

7.  **OpenAI's Codex Updates with Enhanced Computer Use and Memory**
    OpenAI released a major update to Codex, expanding its capabilities beyond coding to include general computer use, web workflows, image generation, and memory. Codex can now operate a computer, remember user preferences, learn from past actions, and handle ongoing tasks, enhancing the agent's operational persistence and interaction fidelity.

8.  **World ID 4.0 Introduces "Proof of Personhood" for AI Agents**
    World, backed by OpenAI CEO Sam Altman, launched World ID 4.0, an update to its digital identity system. This new version provides a "full-fledged proof of personhood" infrastructure for users, businesses, and AI agents, featuring selfie biometric authentication and an "Agent Kit" for managing AI agent identity, delegation, and audit trails.

9.  **Microsoft Agent Framework Reaches Version 1.0 with A2A Protocol Support**
    Microsoft announced the release of version 1.0 of its Agent Framework for both .NET and Python on April 3, 2026. This production-ready release offers enterprise-grade multi-agent orchestration, multi-provider model support, and cross-runtime interoperability, with upcoming support for the Agent-to-Agent (A2A) protocol to enable structured, protocol-driven messaging between agents in different frameworks.
