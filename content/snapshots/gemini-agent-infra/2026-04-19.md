Here's a compact digest of new features, integrations, and announcements from AI agent infrastructure vendors, focusing on protocol and bot-identity features, from April 5, 2026, to April 19, 2026:

1.  **Anthropic Ships Computer Use for Claude**
    Anthropic has released a new "computer use" mode for Claude, enabling the AI agent to browse the web, open files, navigate workflows, and execute tasks on behalf of users. This feature significantly expands Claude's autonomous capabilities for automating real-world work.

2.  **Anthropic Restricts Third-Party Agent Subscriptions**
    Anthropic has implemented a change that prevents Claude subscriptions from being used with third-party agents like OpenClaw. Users are now required to utilize pay-as-you-go API rates for such integrations, potentially leading to increased costs.

3.  **OpenAI SDK Update Introduces Sandboxing for Safer Agent Deployment**
    OpenAI has launched a significant SDK update that includes sandboxing capabilities, designed to enhance the security of AI agents. This allows developers to safely connect frontier models to files and approved tools, streamlining secure deployment for companies.

4.  **Google Releases Agentic Tools for Android Developers**
    Google has introduced new agentic tools for Android developers, aimed at improving efficiency by using 70% fewer tokens and completing tasks three times faster. These tools provide developers with new command-line interface (CLI) skills and access to official knowledge bases for building more intelligent software.

5.  **Google Chrome Skills Integrate Gemini for Persistent Web Automation**
    Google Chrome has launched "Chrome Skills," which integrate with Gemini to allow users to save and reuse AI prompts across various websites. This functionality transforms the browser into a personalized AI workflow engine, enabling persistent automation of repetitive web tasks.

6.  **CrewAI v1.10.1 Adds Native Model Context Protocol (MCP) Server Support**
    CrewAI has released version 1.10.1, which now includes native support for Model Context Protocol (MCP) servers. This update allows role-based agents within CrewAI to invoke and manage MCP integrations, enhancing multi-agent orchestration capabilities.

7.  **Microsoft Launches Agent Framework 1.0.0**
    Microsoft has released Agent Framework 1.0.0, which redefines how developers construct AI agents by separating agent control from the application. This new architectural approach simplifies connecting agents to pre-configured systems within Azure, aiming to reduce technical friction in adoption.

8.  **OpenAI Codex Enterprise Offers MCP and Multi-Agent Workflows**
    OpenAI has launched Codex Enterprise, providing dedicated Codex-only seats for ChatGPT Enterprise users. This offering includes Model Context Protocol (MCP) support, multi-agent workflows, and deep integration with enterprise systems, featuring token-based billing and no rate limits.

9.  **Agent SRE 3.1.0 Public Preview for Autonomous Agent Reliability**
    Agent SRE 3.1.0 has entered public preview, focusing on reliability engineering for autonomous agent systems. This update introduces Service Level Objectives (SLOs) and error budgets tailored to agent performance, chaos testing for resilience, progressive delivery for safer deployments, and real-time monitoring of agent health.

10. **VerifiMind-PEAS Deploys MACP v2.2 Coordination Tools**
    VerifiMind-PEAS has deployed three new Multi-Agent Coordination Protocol (MACP) v2.2 coordination tools as part of its Pioneer-gated premium tier. These tools, `coordination_handoff_create`, `coordination_handoff_read`, and `coordination_team_status`, facilitate enhanced multi-agent collaboration and management.

11. **OpenClaw 2026.4.9 Enhances Multi-Agent Framework with MCP and Role-Based Agents**
    OpenClaw has updated its multi-agent trading framework to version 2026.4.9, introducing enhanced capabilities such as autonomous web scraping of market feeds and role-based agent decomposition (analyst, executor, risk monitor). The update also features Model Context Protocol (MCP)-compatible orchestration and agent state persistence.
