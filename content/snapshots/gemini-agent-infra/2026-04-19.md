Here's a compact digest of new features, integrations, and announcements from AI agent infrastructure vendors, focusing on protocol and bot-identity features, from the last 14 days (April 6, 2026, to April 19, 2026):

1.  **Anthropic Introduces Enhanced Agent Capabilities and Cybersecurity Safeguards with Claude Opus 4.7**
    Anthropic has released Claude Opus 4.7, an updated model that demonstrates stronger performance in coding, agentic tasks, vision, and multi-step operations, with improved thoroughness and consistency. This release also marks the introduction of safeguards designed to automatically detect and block requests indicating prohibited or high-risk cybersecurity uses, alongside a new Cyber Verification Program for security professionals.

2.  **Anthropic Launches Project Glasswing to Secure Critical Software with AI**
    Anthropic announced Project Glasswing, a collaborative initiative with major tech companies including Amazon Web Services, Apple, Google, and Microsoft. This project aims to secure critical software for the AI era by leveraging Anthropic's frontier model, Claude Mythos Preview, to identify and exploit software vulnerabilities for defensive purposes.

3.  **Browserbase Introduces Agent Identity with Web Bot Auth and World ID Integration**
    Browserbase has launched new agent identity features, moving beyond traditional "stealth" methods. This includes the introduction of Web Bot Auth, an open standard that enables agents to present a cryptographically signed proof of their identity when browsing the web. Furthermore, Browserbase agents can now carry a World ID, which provides benefits such as reduced blocking and friction when interacting with websites, and has integrated with partners like Cloudflare, Fingerprint, and 1Password to establish a verifiable and consistent presence for agents online.
